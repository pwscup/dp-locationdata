POI_TK.csv contains POI data in Tokyo (longitude:139.68-139.3, latitude:35.65-35.75) in the SNS-based People Flow Data (nightley) with 8 columns, which are:
1 (poi_id): POI ID
2 (y): Latitude
3 (x): Longitude
4 (category): POI category (滞在者カテゴリ(大分類)/滞在者カテゴリ(小分類)/滞在者カテゴリID)
5 (y_id): y-axis ID (0-31)
6 (x_id): x-axis ID (0-31)
7 (2D_id): region ID (0-1023)
8 (count): Total number of visits by all users
